# Nafaa

Marketplace moderne reliant clients, vendeurs, livreurs et administrateurs.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Prisma
- SQLite en developpement
- Auth.js pret a brancher
- Cloudinary, Socket.io, Leaflet, Recharts

## Lancer

```powershell
npm install
npx prisma migrate dev --name init
npx prisma db seed
npm run dev
```
